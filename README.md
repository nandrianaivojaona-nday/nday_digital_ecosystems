# 🌍 NdaY' Website 4.0

> **Bridging Innovation and Community for a Sustainable Future**

## Overview

**NdaY' Website 4.0** is the official web portal of **NdaY' Individual Enterprise**, serving as the digital gateway to the **NdaY Federated Digital Public Infrastructure (DPI) Ecosystem**.

More than a corporate website, this project introduces the vision, services, and technology foundation that enables communities, local governments, development partners, businesses, and citizens to collaborate through interoperable digital platforms.

The NdaY ecosystem is designed to grow progressively:

> **Community → Commune → District → Region → Nation**

while remaining modular, scalable, secure, and cloud-native.

# Cloud Strategy

The ecosystem is intentionally designed to maximize the use of cloud free tiers during its early growth.

Primary cloud services include:

* GitHub
* Vercel
* Firebase
* Cloudflare

This approach minimizes operational costs while allowing progressive migration toward enterprise cloud services as adoption increases.

# Multi-Tenant Design

The ecosystem is designed as a multi-tenant platform.

One platform can securely serve:

* Multiple Communes
* Multiple Regions
* National Institutions
* Development Partners
* NGOs
* Cooperatives
* Private Operators

Each tenant shares the same platform while maintaining complete data isolation and role-based access control.
